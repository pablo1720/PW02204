Introducción.

Los sistemas operativos basados en Unix disponen de un intérprete de comandos u órdenes (conocido como terminal, consola o shell) que hace de interfaz entre el usuario y el propio sistema operativo. Es decir, mediante la terminal o consola se puede acceder al sistema operativo sin utilizar la interfaz gráfica y realizar todo tipo de tareas en modo texto.

Por defecto, en Ubuntu cuenta con Bash (Bourne-again shell) como intérprete de comandos, el cual cumple muy bien su función de facilitar la interación entre el usuario y el SO, proporcionando un lenguaje de programación intuitivo y sencillo. Sin embargo en términos de personalización y mejora de herramientas se encuentra limitado

Z shell (o simplemente zsh) es un potente intérprete de comandos para sistemas operativos de tipo Unix; a diferencia de Bash, Zsh se diseñó para poder usarse interactivamente. Se le han incorporado muchas de las características principales tanto de otras shells como propias.

¿Por qué Zsh sobre Bash?

Zsh nos ofrece muchas ventajas, como:

  • Eficiencia
  • Completado de tabulador mejorado
  • Expansión de nombres de fichero mejorada
  • Manejo de arrays intuitivo
  • Totalmente personalizable

¿Otra ventaja de Zsh?, cuenta con el framework Oh My Zsh!, el cual gestiona de manera intuitiva, los temas de personalización de la consola, permitiendo una mejor experiencia los usuarios y el sistema operativo.

instalación.

Para la instalación de la terminal Zsh se hará uso de:

  • Sistema operativo GNU/Linux, distribución Ubuntu.
  • Sistema de control de versiones Git 
                            
    sudo apt-get install git
  • Intérprete de comandos basado en la trasferencia de archivos CURL 
                            
    sudo apt-get install curl

En dado caso no se cuente con la distribución Ubuntu, el desarrollo de la instalación puede basarse en el repositorio de RobbyRussell.

Instalación de Zsh. 

                
    sudo apt-get install zsh

Instalación de Oh My Zsh! 

                
    sh -c "$(wget -O- https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

Se debe aceptar (opción "y") la instalación de zsh como shell por defecto.

Zsh por defecto.

En este punto ya se puede apreciar como la terminal cambia estéticamente. Aunque si se cierra y vuelve a abrir volverá al aspecto que se tenía previamente.

Para evitar esto, dentro de las preferencias de la terminal, se debe asignar zsh como intérprete por defecto.

Dentro de la barra de menús, se selecciona la opción Editar -> Preferencias.

En Perfiles, se selecciona el perfil que utiliza la terminal, en este caso el por defecto.

Se ingresa a la pestaña Comando, y se selecciona la opción Ejecutar una orden personalizada en vez de mi intérprete. Se habilitará un campo de texto, donde se debe ingresar el comando zsh

Esto hará que al iniciar la terminal se ejecute el comando zsh, iniciando dentro del sistema el intérprete zsh.

Temas.

Luego de asignar zsh como shell predeterminada, se procede a la gestión de los recursos y herramientas que se harán uso en la misma.

Uno de estos rubros es la personalización del aspecto o tema de la terminal, lo cual se logra a través del archivo .zshrc. Debido que el tema de estilos se trata de algo muy personal, en el repositorio Themes de RobbyRussell, se detalla la lista de temas y sus wikis, con el fin de ampliar las posibilidades de personalización en este ámbito.

Para este caso se hará uso del tema agnoster; para ello se debe ingresar desde cualquier editor de texto al archivo ./zshrc, y se realizan las siguientes modificaciones. 

                
    sudo nano ~/.zshrc

Una vez realizados los cambios en el archivo, se deben aplicar los cambios en el intérprete. Esto se logra a través del siguiente comando: 

                
    source ~/.zshrc

Como es de notar la apariencia ha cambiado pero no se adapta completamente. Esto se debe a que falla la tipografía que el tema usa por defecto, por lo que esta se debe agregar. 

                
    curl -L https://github.com/hbin/top-programming-fonts/raw/master/install.sh | bash

Al instalar la familia fuentes, se debe cambiar la fuente predeterminada de la terminal; esta debe ser de la familia Menlo.

Instalación de Fuzzy Finder.

Otra herramienta primordial a la hora de trabajar con la terminal es un historial, nativamente se puede usar el comando ctrl + r para realizar una reverse-search, pero existe una alternativa potente llamada fzf (fuzzy finder). Para su instalación se ingresan los siguientes comandos: 

                
    git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf

    ~/.fzf/install

Se aceptan todas las condiciones y se reinicia el intérprete. A partir de este punto al realizar ctrl + r, se accederá a un historial interactivo desde el cual se prodrá navegar dentro haciendo uso de las flechas direccionales del teclado. 

                
    source ~/.zshrc
