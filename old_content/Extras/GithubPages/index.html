Uso de Github pages.

Hosting integrado a la plataforma.

Implementación

GithubPages es un servicio que proporciona la plataforma GitHub para poder alojar sitios web estáticos. Para logra integralo a cualquier proyecto se debe seguir los siguientes pasos.

  1. Si no se tiene repositorio creado, se debe inicializar uno. Para esta práctica se hará uso del repositorio "githubPagesExample".

  2. Luego se debe agregar toda la estructura deseada del sitio web estático. Solo hay tener en cuenta que debe de existir un archivo index.html como punto de entrada para la aplicación.

  3. A continuación se debe configurar el servicio de githubPages para este repositorio en concreto. Para ello se debe ingresar a la sección de Settings -> Options -> Github Pages.
