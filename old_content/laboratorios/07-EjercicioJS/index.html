Ejercicio guiado

Problema a resolver.

Haciendo uso de JavaScript, diseñe la lógica de un registro de asitencia para los laboratorios. Los campos a tomar en cuenta son el carnet, el horario de laboratorio al que pertenece, y si la persona llegó tarde o no.

Cada persona ingresada debe de mostrarse en una tabla como registro; se debe asegurar que el carnet ingresado tenga el formato correcto, de lo contrario el botón de "ingresar" deberá estar desailitado.

Por cada registro se debe llevar el control de la fecha y hora a la que este fue igresado al sistema. Esta puede ser mostrada en cuanquier formato.

Para la estructura HTML del registro, puede basarse en el siguiente documento. Se debe tomar en cuenta que se ha utilizado el framework Bootstrap para estilizar el sitio web.
