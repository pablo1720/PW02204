Recrear página

Con la siguiente imagen: Recrear la página mostrada haciendo uso de HTML5 y CSS. Se deberán seguir los siguiente lineamientos:

  • Para el título: hacer uso de la fuente sans-serif.

  • Para los parrafos: hacer uso de la fuente Roboto.

  • El color de fuente para los parrafos es #7f888f

  • El texto en el botón "LEARN MORE" debe cambiar color al pasar el mouse sobre el.

  • El codigo del color rojo de la página es #f56a6a.

  • Los íconos deben ser de Font Awesome

  • La imagen puede ser cualquiera de su elección.
