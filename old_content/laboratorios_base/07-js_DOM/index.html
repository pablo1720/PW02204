Árbol DOM

Al momento que un navegador carga una página web, este crea un modelo entero de la página. Este modelo es comunmente conocido como Arbol DOM el cual se almacena en la memoria del navegador.
El Arbol DOM conciste en cuatro principales nodos:

Document Node

Toda página creada con HTML tiene una forma establecida llamada: DOM tree. Todos los elementos, atributos y texto son un DOM node.
La raiz del árbol es el nodo representa la pagina entera conocida como document node.

Cuando se accede a cualquier elemento, atributo o texto, se debe navegar por medio de él

Attribute Node

Al crear un elemento HTML este pueden contener atributos, los cuales dentro del árbol DOM se conocen como Attribute Node.

En la Ilustración 1, el elemento <div> tiene un atributo, el cual no es hijo del div, sino que, tanto la etiqueta <div> y el atributo asociado, conforman el elemento HTML

Element Node

Son todos los elementos HTML que describen la estructura de una página

Text Node

Es el texto que se encuentra dentro de un elemento HTML. (Por ejemlplo: <em>, <p>, <li>, entre otros elementos que tienen como hijo un texto);

Acceder a elementos HTML.

Al momento de acceder a un elemento del árbol DOM, se puede realizar de las siguientes maneras:

Seleccionar multiples elementos (NODELIST).

Todas los siguientes metodos devuelven listas de elementos que se acceden como si de un arreglo se tratasen.

getElementsByClassName().

Recibe como parámetro el nombre de la case css de los elementos que se desean capturar.
Ejemplo:

document.getElementsByClass-Name("success")

getElementsByTagName().

Recibe como parámetro el nombre de la etiqueta HTML y retorna todos los elementos que cumplan la condición.
Ejemplo:

document.getElementsByTag-Name("li")

querySelectorAll().

Recibe como parámetro el selector CSS, retorna todos los elementos que cumplan con la condición.
Ejemplo:

document.querySelectorAll
("li.item-nav")

Seleccionar un elemento individualmente.

getElementById()

Recibe como parámetro el nombre que se le ha asignado al atributo id del elemento.
Ejemplo:

document.getElementById("nav-brand")

querySelector()

Recibe como parámetro un selector css y retorna el primer elemento que cumpla con la condición.
Ejemplo:

document.querySelector
("li.nav-item")

Modificar HTML con JavaScript

Obtener / Editar texto de un elemento

Existen muchas maneras diferentes de obtener / editar el valor de un elemento en específico, se mostrarán los métodos más comunes:

innerHTML

Devuelve el contenido con formato HTML de un elemento:
Sintaxis:

var text = document.getElementById("brand")
text.innerHTML // Obtiene contenido en formato HTML

textContent

Devuelve el texto sin formato de un elemento y todos sus decendientes.
Sintaxis:

var text = document.getElementById("brand")
text.textContent // Obtiene texto

createElement()

Crea un elemento HTML.
Sintaxis:

var listItem = document.createElement("li")// Crea el siguiente elemento <li ></li >

createTextNode()

Crea un nodo de texto
Sintaxis:

var text = document.createTextNode("texto cualquiera")// Crea un nodo de texto

appendChild()

Recibe un nodo y le añade ese nodo como hijo al elemento actúal
Ejemplo:

var title = document.createElement("h2")
var text = document.createTextNode("texto cualquiera")// Crea un nodo de texto
title.appendChild(text)

Consultar: MDN web docs para mayor información.

Obtener / Añadir atributos a elementos HTML

getAttribute()

Recibe como párametro un atributo y retorna el valor del atributo ingresado, si el elemento no tiene el atributo deseado, retornará nulo. Ejemplo:

var div = document.getElementById("main-content")
div.getAttribute("style")// Retorna una cadena de texto con el valor del atributo style en el HTML

hasAttribute()

Recibe como párametro un atributo y retorna verdadero o falso. Ejemplo:

var div = document.getElementById("main-content")
div.hasAttibute("style")// true or false

setAttribute()

Recibe como párametro un atributo y su valor; al elemento que se le aplica este método, le añade los valores enviados. Ejemplo:

var div = document.getElementById("main-content")
div.setAttribute("style","display:none")// Ocultara el elemento HTML con el id "main-content"

removeAttribute()

Recibe un atributo y lo eliminara del elemento al que se le aplique. Ejemplo:

var div = document.getElementById("main-content")
div.removeAttribute("style")// Elimina el atributo style

cassName

Propiedad de un elemento, obtiene o edita el atributo class. Ejemplo:

var div = document.getElementById("main-content")
div.className = "success-class"// Añade la clase "success-class" al elemnto con id "main-content"
var class = div.className // Guarda el valor del atributo class

id

Propiedad de un elemento, obtiene o edita el atributo id. Ejemplo:

var div = document.getElementById("main-content")
div.id = "main section"// Agrega el atributo id con el valor "main-section"
var idName = div.id // Guarda el valor del atributo id

Consultar: MDN web docs para mayor información.

Eventos

Cuando el navegador carga una página web y el usuario interactúa con ella, el navegador registra diferentes tipos de eventos, y al ocurrir ciertos eventos un script puede reaccionar ante ellos.

Puedes consular MDN web docs para ver la lista completa de eventos que pueden utilizarse.

Se pueden utilizar de muchas maneras; añadiendo un atributo al elemento directamento en el HTML o sea dentro de un script, capturando el elemento y asignandole un evento o añadiendo un listener

Añadir evento a elemento dentro del HTML

Se recomienda no utilizar este método ya que se concidera una mala practica, sin embargo, es necesario conocerla en caso se encuentren en algunos sitios. Ejemplo:

Código HTML
<input id = "input-pass" type = "text" onkeyup= "verifyLength()" />
<label id = "alert-content"/>...
Script
function verifyLength( ) {
var input = document.getElementById("input-pass");
var msg = document.getElementById("alert-content");
if(input.value.length < 5){
msg.textContent = "La contraseña debe ser mayor a 5 caracteres";
}

}

Cuando el elemento input-pass detecte que pcurre un evento onkeyup, es decir, que una tecla se suelta luego de ser pulsada, se ejecutará la función verifyLength().

Manejar eventos por medio de DOM.

Es una de las maneras modernas y mas habituales de ejecutar una función al ocurrir un evento, en este metodo, todo el funcionamiento se codifica dentro de las etiquetas <script></script> o dentro de un archivo .js

Sintaxis:

element.onevent = functionName;

Ejemplo:

Script
function verifyLength( ) {
// Se define y se crea la función

}
var element = document.getElementById("input-first"); // Se obtiene el elemento
element.onkeyup = verifyLength;

Usar event listeners

Usar listeners es la manera mas reciente de manejar eventos, con este método se puede manejar mas de una función a la vez.

Sintaxis:

element.addEventListener('event',functionName [,Boolean])
El primer parametro del método addEventListener(), es el evento que se desea escuchar
El segundo parametro el la función que se ejecuta al ocurrir el evento
El último es llamado fase de captura usualmente colocado como false
Mas información sobre el tercer párametro en: El tercer parámetro de addEventListener

Ejemplo:

function verifyLength( ) {
// Se define y se crea la función

}
var element = document.getElementById("input-first"); // Se obtiene el elemento
element.addEventListener('keyup',verifyLength,false)
En el párametro de la función a ejecutar, puede definirse como anonima.
